Project Beagle: Hacking A Design System Component Library

Join this squad to set up a contribution docs site to enable aspiring designers and devs to join open-source project workathons.

Level of Difficulty: Intermediate
Role: Designer, React Developer


This feature is named after the Beagle.

Beagle, small hound dog breed popular as both a pet and a hunter. It looks like a small foxhound and has large brown eyes, hanging ears, and a short coat, usually a combination of black, tan, and white. The Beagle is a solidly built dog, heavy for its height. It generally excels as a rabbit hunter and is typically an alert, affectionate dog.
